# Tempo Card

## App Name

Tempo Card

## Tagline

Post your chain rhythm

## Description

Publish a BPM, mood, and short note on Base to create a collectible rhythm card with a clean player-style layout.

## Category

Music / identity / creative social

## Short Pitch

Tempo Card gives people a lightweight way to post a track-like identity card on Base, turning mood, tempo, and a short note into a clean onchain keepsake.

## Production Environment Variables

- `NEXT_PUBLIC_TEMPO_CARD_CONTRACT_ADDRESS`: deployed tempo card contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
